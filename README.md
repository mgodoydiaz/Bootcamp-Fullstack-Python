# Bootcamp Fullstack Python Developer

Repositorio de evaluaciones finales del curso **Fullstack Python Developer**, dictado por **Kibernum** en conjunto con **SENCE**.

## Descripción

Este repositorio contiene las evaluaciones finales de cada módulo del bootcamp. Cada carpeta corresponde a un módulo distinto del programa de formación.

## Módulos

| Módulo | Descripción |
|--------|-------------|
| M2 - Fundamentos de Frontend | Evaluación: CV online disponible [aquí](https://mgodoydiaz.github.io/cv_online/) |
| M3 - Python Básico | Evaluación final: Dr Python (Sist. interactivo por consola) |
| M4 - Python Avanzado | Evaluación final: Sist. Gestión de Jugadores de Fútbol (POO) |
| M5 - Bases de datos | Evaluación final: Base de datos de una clínica veterinaria |

## Tecnologías

- Python
- HTML / CSS / JavaScript
- Bases de datos (PostgreSQL)
