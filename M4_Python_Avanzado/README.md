# Evaluación Modular 4

## Sistema de Gestión de Jugadores de Fútbol

### Descripción

Desarrollo de un sistema en Python para gestionar jugadores del club
Python FC, aplicando Programación Orientada a Objetos, herencia, manejo
de archivos y control de excepciones.

### Objetivos Técnicos

-   Definir clase Jugador con atributos nombre, edad, posición y goles.
-   Implementar método constructor **init** y mostrar_info().
-   Crear subclase Capitan con atributo liderazgo.
-   Aplicar herencia y sobrescritura de métodos.
-   Guardar y cargar jugadores desde archivo de texto.
-   Implementar manejo de errores con try/except.
-   Validar rangos de edad y goles mediante excepción personalizada.

### Estructura

-   Clase RangoInvalido
-   Clase Jugador
-   Clase Capitan
-   Clase Jugadores para gestión y persistencia
-   Archivos de prueba Chile vs Argentina y Chile vs Uruguay 2015

### Ejecución

Ejecutar:

python M4_evaluacion_modular.py

Descomentar en el bloque principal la función que se desee probar.
